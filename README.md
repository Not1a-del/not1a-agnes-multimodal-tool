# Not1a · Agnes 多模态工具

一个本地运行的多模态 API 调试工具。它面向 OpenAI 兼容接口和常见中转接口，提供图片生成、图片编辑/图生图、视频生成、异步任务查询、聊天/多模态聊天测试、自定义供应商、模型列表自动分类和本机参考图上传。

> 本项目默认在本机启动服务。API Key 只保存在你的浏览器本地存储中，并由本机服务转发请求，不会内置在项目文件里。

## 功能

- 图片生成：支持文生图、图生图/图片编辑、尺寸、质量、数量、种子等参数。
- 视频生成：支持创建视频任务、自动轮询任务进度、完成后收集视频链接。
- 聊天测试：气泡式聊天界面，支持文字和图片一起发送。
- 自定义接口：可新增、命名和保存多个接口地址与 Key。
- 模型识别：可读取 `/models`，并按 Agnes、Grok、DeepSeek、GLM、Gemini、Qwen、GPT 等系列折叠展示。
- 本机参考图：电脑端可选文件，移动端可从相册选择，自动转成可发送的图片数据。
- 完成列表：自动保存已生成图片/视频链接，短显示但打开时使用完整链接。
- 本地代理：通过本机 `server.py` 转发请求，尽量减少浏览器跨域限制。

## 快速开始

### Windows

1. 下载并解压项目。
2. 双击运行：

```text
web_tool\start_web_tool.bat
```

3. 浏览器会打开：

```text
http://localhost:8765
```

4. 在页面中填写接口地址和 API Key，然后选择图片、视频或聊天栏目开始测试。

### 手动启动

```powershell
cd web_tool
python server.py
```

然后打开：

```text
http://127.0.0.1:8765
```

## 预设接口

工具内置了几个常用供应商预设：

- Agnes / APIHub
- Grok / xAI
- GPT Image / OpenAI
- 自定义 OpenAI 兼容接口

自定义接口填写 URL 和 Key 后，可点击“获取模型”读取模型列表。工具会尽量按关键词把模型分到图片、视频、聊天三类，并排除 embedding、rerank、vector 等向量类模型。

## 隐私说明

- 项目不包含任何 API Key。
- 页面保存的接口地址、Key、自定义供应商和生成记录都在你自己的浏览器本地。
- 本地服务默认只监听 `127.0.0.1:8765`，用于静态页面访问和请求转发。
- 如果你要截图或分享页面，请先确认没有暴露自己的 Key 或敏感接口地址。

```powershell
python build.py
```

会在桌面生成：

- `Not1a_Agnes_多模态工具_单文件版.html`
- `Not1a_Agnes_多模态工具_GitHub本地服务版.zip`

GitHub 推荐提交源码；Release 附件可以放这个 zip。
