# 更新记录

## v1.0.2

- 优化生成等待反馈，图片、视频和任务查询会在请求期间保持可见的生成动画。
- 视频任务轮询开始后会展示原比例占位，避免长时间排队时页面看起来没有变化。
- 请求成功但未识别到图片或视频链接时，会提示查看日志，而不是留下空白结果区。
- 清理脚本中的重复函数和未使用函数，降低后续维护成本。

## v1.0.1

- 新增“复制脱敏 cURL”，便于反馈接口问题。
- 新增“下载 JSON”和“复制问题报告”，方便记录请求结果并脱敏反馈。
- 清理 GitHub 首页说明，不展示本地维护打包信息。
- 保持发布包不包含 API Key 和已生成文件。

## v1.0.0

- 支持图片生成、图生图/图片编辑、视频生成、任务查询和聊天测试。
- 支持 Agnes、Grok、GPT Image / OpenAI 和自定义 OpenAI 兼容接口。
- 支持读取模型列表并按系列折叠展示。
- 支持本机参考图上传和已完成项目列表。
