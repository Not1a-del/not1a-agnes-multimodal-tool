Not1a · Agnes 多模态工具 使用教程

一、这是什么

这是一个本地网页工具，用来测试多模态模型接口。你可以在页面里填写接口地址和 API Key，然后进行图片生成、图片编辑/图生图、视频生成、视频任务查询和聊天测试。

二、怎么启动

Windows 用户：

1. 解压整个压缩包。
2. 进入 web_tool 文件夹。
3. 双击 start_web_tool.bat。
4. 浏览器会自动打开 http://localhost:8765。

如果浏览器没有自动打开，可以手动打开：

http://localhost:8765

三、需要填写什么

1. 供应商预设：可以选择 Agnes / APIHub、Grok / xAI、GPT Image / OpenAI，或新建自定义 API 接口。
2. 接口地址：例如 https://apihub.agnes-ai.com/v1。
3. API Key：填写你自己的 Key。
4. 模型：可以手动输入，也可以点击“获取模型”自动读取并分类。

四、主要功能

1. 图片生成
   - 支持文生图。
   - 支持图生图/图片编辑。
   - 参考图可以填写 URL，也可以从本机上传。

2. 视频生成
   - 提交任务后会自动轮询。
   - 等平台返回完整视频链接后，页面会自动展示。
   - 支持参考图 URL，也支持从本机上传参考图。

3. 聊天测试
   - 使用气泡式聊天界面。
   - 支持文字消息。
   - 支持本机上传图片或填写图片 URL 后一起发送。

4. 已完成项目
   - 生成完成的图片和视频链接会自动保存到页面底部。
   - 页面只短显示链接，点击“打开完整链接”时会跳转到真实完整链接。

五、Key 会不会泄露

工具不会内置任何 Key。你填写的接口地址、Key、自定义接口和历史记录，只保存在你当前浏览器的本地存储里。本地服务默认只在你自己的电脑上运行。

六、发给别人怎么用

把压缩包发给对方。对方解压后进入 web_tool 文件夹，双击 start_web_tool.bat，然后填写自己的接口地址和 API Key 即可。

七、常见问题

1. 页面打不开
   - 确认电脑安装了 Python。
   - 确认没有其他程序占用 8765 端口。

2. 请求失败
   - 检查接口地址是否包含 /v1。
   - 检查 API Key 是否正确。
   - 检查模型名是否存在。
   - 点击“查看日志”看接口返回的完整错误。

3. 上传本机图片后接口不接受
   - 工具会把本机图片转成 data:image/...base64 形式。
   - 少数接口不支持这种格式，这时请改用图片 URL。

4. 视频一直排队
   - 视频接口通常是异步任务，平台排队时进度可能长时间不变。
   - 工具会自动等待；如果平台最终返回视频链接，就会展示完整视频。
